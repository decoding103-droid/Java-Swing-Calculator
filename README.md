# Java Swing Calculator

A simple desktop calculator application built using Java Swing and IntelliJ IDEA.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear Button
- Interactive GUI

## Technologies Used

- Java
- Java Swing
- IntelliJ IDEA

## What I Learned

- Object-Oriented Programming (OOP)
- Event Handling with ActionListener
- GUI Development using Java Swing
- Desktop Application Development

## Future Improvements

- Dark Mode
- Decimal Support
- Backspace Button
- Scientific Functions
- Calculation History
